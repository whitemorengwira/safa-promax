# SA Film Academy · Getty Images Prompt Directory

This directory contains 200+ specific search prompts for the Getty Images library to populate the SA Film Academy website.

## **Core Casting & Style Rules:**
- **Casting:** Black South African youth (18-30 years old).
- **Style:** Cinematic, high-end editorial, low-key lighting with warm gold accents.
- **Environment:** Professional film sets, modern tech workspaces, high-end studios.
- **Tone:** Professional, focused, aspirational, and authentic.
- **No-Go:** Corporate stock aesthetics, forced smiles, or movement-themed text.
