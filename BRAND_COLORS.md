# SA Film Academy Brand Colours

**Source inspected:** [safilmacademy.org](https://safilmacademy.org/)  
**Inspection date:** 20 May 2026

The official SA Film Academy website uses **red as the primary institutional accent colour**. Its generated CSS declares `--primary_color:#cc0000`, red menu hover treatments, red button gradients, red link hover states, and a red top border for the secondary header. The ProMax site therefore now treats red as the dominant brand colour, with gold retained only as a secondary cinematic highlight.

| Token | Hex | Usage |
|---|---:|---|
| Primary red | `#CC0000` | Main CTA backgrounds, menu hover states, active navigation, section accents |
| Link red | `#C10000` | Text links and subtle accent strokes |
| Deep red | `#690E0E` | Dark red shadows, deep gradients, premium editorial depth |
| Dark red | `#9D1111` | Button borders, darker interaction states |
| Vivid red | `#FE2020` | Small highlights and glow effects only |
| Alert red | `#DC3232` | Secondary red treatment where brighter UI feedback is needed |
| White | `#FFFFFF` | Primary text on red and dark surfaces |
| Charcoal | `#333333` | Official site dark text reference, adapted into ProMax dark surfaces |
| Secondary green | `#65BC7B` | GreenSet-related secondary accent only, not a dominant SAFA colour |
| Legacy gold | `#D4A574` | Retained sparingly for cinematic prestige, borders, and heritage emphasis |

## Implementation Notes

The ProMax website should preserve its black cinematic environment while making **SAFA red** the most visible brand signal across navigation, buttons, underlines, hover states, gradients, data accents, and image overlays. Gold must no longer compete as the main brand colour. It should act as a restrained editorial highlight.

> Brand rule: use **red first**, gold second, black as the cinematic base, and white/off-white for legible British-English editorial text.
